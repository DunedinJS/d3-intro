D3 Introduction

What is D3?

D3 is a JavaScript library which can use data to manipulate HTML and SVG. This makes it a powerful tool for building data visualisations.

D3 does not provide a standard set of charting options 'out of the box'. Instead it provides lower-level functions for data transformation, physics, and HTML or SVG manipulation.

Although D3 has a somewhat steep learning curve this can be attributed to its specialised purpose which is quite unlike any other JavaScript libraries. With some basic familiarity it is a greatly expressive and efficient library to work with.

Chart 1 — HTML bar chart

This is a simple demo to see D3 in action. Here we draw a simple bar chart with HTML elements.

Hover over the bars to see their values.

Inspect this chart with your browser's developer tools.
You will see that the #chart1 element contains a separate <div> for each of the eight random values in the data set and their heights are determined by those values.

Also see the documented source code in source/chart1.js

An issue which you might have noticed in Chart 1 is that the chart height relates directly to the largest data value.

For example in source/chart1.js, change const max = 1000;.
You will see that the chart is rendered much taller because the values are much larger numbers.

This leads us into Chart 2 which introduces scales.

Chart 2 — HTML bar chart with scaling

A simple HTML bar chart with scaling of the bar heights.

A linear scale is used here to determine the bar heights. No matter what the largest value is, its bar will have a height of 100 pixels.

D3 also provides other scale types such as time, and logarithmic.

Hover over the bars to see their values.

See the documented source code in source/chart2.js

Chart 3 — SVG line chart

A simple line chart which plots values over time. This renders an inline SVG and draws a path from the data. SVG graphics are used because they offer much more visual features than plain HTML elements.

A linear scale is used here to scale the X axis to the chart height and a time scale is used to scale Y axis to the chart width.

The path drawn with a straight lines between points but it is also possible to specify other curve functions. See the D3 curves documentation for details.

See the documented source code in source/chart3.js

Chart 4 — SVG heatmap

A heatmap which plots values over time.

D3 provides an extensive set of interpolations functions for blending between to values. Here we use d3.interpolateRgb to determine which colour each tile should be based on its value.
See the D3 interpolate documentation for details.

See the documented source code in source/chart4.js
